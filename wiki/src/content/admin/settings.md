# Einstellungen

Im Administrationsbereich (ACP) kannst du die globalen Einstellungen für die Match-Engine, Vereine und Wettbewerbe konfigurieren.

## Match-Engine Einstellungen
Ganz wichtig für das Balacing ist die Anpassung der Events Wahrscheinlichkeiten:

- **Tor-Wahrscheinlichkeit:** Definiert wie oft Torchancen entstehen.
- **Karten-Wahrscheinlichkeit:** Wie streng der Schiedsrichter generell ist.
- **Verletzungsrisiko:** Prozentuale Wahrscheinlichkeit, dass sich ein Spieler in einem Zweikampf verletzt.

Diese Settings findest du im Tab "Match Engine" deines Admin Dashboards.
